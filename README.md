全国城市坐标经纬度

## 字段说明

|     字段名      |        描述        |
| :----------: | :--------------: |
|     `id`     |        ID        |
|    `pid`     |       父ID        |
| `shortname`  |        简称        |
|    `name`    |        名称        |
|  `fullname`  |        全称        |
|   `level`    | 层级；1省;2市;3(区\|县) |
|  `pingyin`   |       简称拼音       |
|    `code`    |       长途区号       |
|  `zip_code`  |       邮政编码       |
|   `first`    |      名称首字母       |
| `longitude` |        经度        |
|  `latitude`  |        纬度        |
